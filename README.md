# Circle Fun

This little React application allows you to control how the canvas renders a circle. 

Depencies:
* NodeJS

To use this app all you have to do is:

1. Clone the repo.
2. cd into the root folder.
3. run `npm install`.
4. run `npm run dev`.
5. The app will be avialable at `http://localhost:5173/`
